# Proyecto: Control de Versiones y Automatización
Repositorio de ejemplo desarrollado para la experiencia educativa de Documentación de Proyectos de Tecnologías Computacionales.

## Git ejemplo
Repositorio de ejemplo para practicar Git, ramas, merges y commits.

## Archivos del Proyecto
* **main.py**: Primer archivo del proyecto, creado y modificado en la rama principal.
* **prueba1.py**: Archivo donde se realizó el primer commit.
* **prueba2.py**: Archivo donde se realizó el segundo commit.
* **nuevo.py**: Archivo para guardar constancia de la creación del script.
* **iniciar_ssh.sh**: Script Bash para la inicialización automática del agente SSH
